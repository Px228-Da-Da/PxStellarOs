### INSTALL:
1. Запустіть `install_Lib.bat`.
2. Запустіть `setup.py` (або `python setup.py` у терміналі).



### Пояснення:
- **6-та цифра:**  
  - `0` — зміни в коді.  
  - `1` — зміни в тексті.  
  - `2` — зміни в графіці.  
  - `3` — інші ресурси.  
- **7-ма цифра:**  
  - `0` — можливі баги.  
  - `1` — багів немає.
